# Contributing to Awesome Libya

We love contributions from the community!

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your feature (`git checkout -b awesome-libya`)
3. **Edit** the `README.md` to add your link or resource
4. **Commit** your change (`git commit -m "Added: awesome-libya"`)
5. **Push** to your fork and submit a **Pull Request**

## Contribution Rules

- Add only **Libya-related** or **Libyan-created** resources
- Keep entries sorted alphabetically
- Follow the markdown format and keep descriptions short
- Check for duplicates before submitting
- Use **verified sources** and clean formatting
